# Purrductivity: focus with a cat who's counting on you 🐱

Purrductivity is an adorable productivity extension that pairs a customizable
focus timer with a website blocker guarded by one very sleepy orange cat.

Set your focus length, add your most distracting sites, and start the timer.
While you're focused, those sites are blocked, try to sneak onto one and you'll
find your kitty curled up fast asleep, gently shaming you back to work. 😴

Quit your focus session early? Your cat bolts off the screen in disappointment.
But make it to the end of the timer and you'll be greeted by one very proud, very
happy cat, then asked if you're ready to do it all again.

Cute, motivating, and just guilt-trippy enough to keep you on task. 🐾

## Features

- ⏱️ Customizable focus timer with optional **Strict mode** (no early quitting!)
- 🚫 Personalized website blocker
- 😴 A sleeping cat that blocks distracting sites
- 🏃 A cat who runs away when you give up early
- 🎉 A happy-cat celebration when you finish

## Load the Extension in Chrome

1. Open Chrome and go to **`chrome://extensions`**.
2. Toggle **Developer mode** on (top-right corner).
3. Click **Load unpacked** (top-left).
4. Select the **`purrductivity`** folder (the folder itself, not a file inside it).
5. The 🐱 **purrductivity** card appears. Click the puzzle-piece icon in the
   toolbar and **pin** Purrductivity so the cat stays visible.

> After editing any file, return to `chrome://extensions` and click the **↻ reload**
> icon on the Purrductivity card to pick up your changes.

## Usage

1. Click the **🐱 toolbar icon** to open the popup.
2. Add distracting sites to your **block list** (e.g. `x.com`, `youtube.com`).
3. Set a **focus length** in minutes and click **Start focus**
   (tick **Strict** to disable early stopping for the whole session).
4. Visit a blocked site during focus → you'll meet the **sleeping cat**. 😴
5. Stop early → the **cat runs away** and your sites unblock. 🏃
6. Finish the timer → a **happy cat** celebrates, and asks if you want to go again. 🎉
